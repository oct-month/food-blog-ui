# 美食博客的前台

> 后台见  [FoodBlog](https://github.com/oct-month/FoodBlog)

nodejs 版本为 `v12`

## 安装依赖

```
npm install
```

### 运行服务

```
npm run serve
```

服务工作在：http://localhost:8083/

### 编译生成静态文件

```
npm run build
```

生成的静态文件在 dist/ 目录下
